はじめまして、ひなたです。Webエンジニアとして働きながら学んだことを書いていきます。

## このブログの構成

- **フレームワーク**: Next.js (App Router) の静的エクスポート
- **記事**: MDX（Markdown + React コンポーネント）
- **UI**: Tailwind CSS + shadcn/ui
- **ホスティング**: GitHub Pages

記事は `src/content/posts/*.mdx` に置くだけで一覧に反映されます。

## コードもハイライトされます

```ts
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("world"));
```
